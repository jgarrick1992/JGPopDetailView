## Installation

#### CocoaPods

With [CocoaPods](http://cocoapods.org/), add this line to your `Podfile`.
```
pod 'JGPopDetailView'
```

and run `pod install`, then you're all done!

#### Manually
1. Download and drop ```/JGPopDetailView``` folder in your project.  
2. Congratulations!
