## Installation

#### CocoaPods

With [CocoaPods](http://cocoapods.org/), add this line to your `Podfile`.

```
pod 'PopMenu'
```

and run `pod install`, then you're all done!

#### Manually
1. Download and drop ```/PopMenu```folder in your project.  
2. Congratulations!
